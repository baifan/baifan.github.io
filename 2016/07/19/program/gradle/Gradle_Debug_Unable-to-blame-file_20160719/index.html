Gradle Unable to blame file

在为新的gradle项目，执行sonarqube时，报出了如题的错误Gradle Unable to blame file。

背景

命令

gradle build sonarqube

错误信息

解决方案

因为这是一个新的项目，没有commit过代码。
所以使用

git add .
git commit . -m ‘初始化项目’

在执行上述的命令就没有问题了

gradle build sonarqube

也可以顺带push到远程环境

git remote add git:XXXXXX.git
git push -u origin master
